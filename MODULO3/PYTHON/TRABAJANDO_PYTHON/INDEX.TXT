EJEMPLO 7

Desarrollar un programa que permita:
  - Calcular el área y perimetro de diferentes figuras geometricas 
    (Circulo, Rectangulo, Cuadrado)
  - Mostrar la información de cada figura 
  - Almacenar varias figuras y mostrar un resumen de todas ellas
  
DESVENTAJAS DEL ESTRUCTURADO
----------------------------
Código repetitivo.
Dificil de escalar. (Agregar nuevas figuras)
Datos y lógica juntos.

VENTAJAS DE LA VERSION POO
--------------------------
Código mas organizado.
Reutilizar código.
Mas escalable o extensible.

TEMAS DEL EJEMPLO
-----------------
Clase abstracta: Define los metodos que los hijos implementaran mas no se implementa en el padre
Clase: Plantilla a partir del cual podemos crear objetos
Objeto: Instancia de una clase
Herencia: Reutilizar código
Polimorfismo: Define en el padre un método y cada hijo lo implementa a su manera
Encapsulamiento: private, set y get
Sobrecarga de métodos: Utilizar el mismo nombre para distintos métodos
